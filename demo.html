Adding a New Theme

To add a new theme, simply:

  1. Create a new folder in the assets directory
  2. Add your design images to that folder
  3. Run the build script
# Create a new theme folder mkdir assets/theme4 # Add your images # (copy your design files to assets/theme4/) # Build the HTML npm run build # or node build.js # or double-click build.bat (Windows)
